# Yada

> Browser-native original-IP gacha game platform targeting North America and Western Europe.

## What Is Yada?

Yada is a full-stack browser game platform — not just a single game, but a discovery portal, publishing layer, and developer ecosystem built around original IP titles. The core model is gacha IAP + battle pass monetization, competing directly with G123/CTW Inc. (500M users, licensed anime HTML5 gacha).

The three strategic differentiators:
1. **Original IP** — characters and worlds we own outright, not licensed anime
2. **Fair Play Pledge** — transparent odds, pity systems, F2P-completable content
3. **Western identity** — western-native art direction and cultural design, not Japan-first

## Key Decisions

- **Target market:** North America + Western Europe first (highest ARPU for gacha, clear whitespace in original-IP browser RPG)
- **Monetization:** Gacha IAP (character banners) + Battle Pass as conversion engine
- **Platform scope:** Full stack — discovery, publishing, and third-party developer ecosystem
- **Codename:** "Yada" (working name — real brand TBD)
