## Game Development Terms

| Term | Meaning | Context |
|------|---------|---------|
| GDD | Game Design Document | Full structured doc: mechanics, loop, economy, characters, roadmap |
| IP | Intellectual Property | Original characters, worlds, lore we own outright |
| the flagship | First primary original IP game | Yada's anchor title |
| live service | Ongoing content updates post-launch | 6-week patch cadence target |
| patch cadence | How often the game updates | Target: major update every 6 weeks |
| content velocity | Speed of content production | Critical retention driver |
| loop | Core gameplay loop | The repeating cycle of actions that drives engagement |
| meta | Meta-game layer | Progression systems outside core loop (gacha, crafting, etc.) |
| idle | Idle/AFK game mechanics | Progression while not actively playing |

## Monetization Terms

| Term | Meaning | Context |
|------|---------|---------|
| F2P | Free-to-play | No upfront cost; revenue via IAP/ads |
| IAP | In-app purchases | Primary revenue model for Yada |
| gacha | Randomized loot/character pull system | Core monetization mechanic |
| banner | A gacha pull campaign | Limited-time character or item banner |
| pity | Guaranteed pull after X attempts | Industry standard: hard pity ~90 pulls |
| rate-up | Featured characters with boosted drop rates | Standard gacha mechanic |
| 50/50 | Guaranteed rate-up or standard 5★ mechanic | Player-protection standard post-Genshin |
| whale | High-spending player | >$100/month; drives 50-70% of revenue |
| dolphin | Mid-spending player | $20-100/month |
| minnow | Low-spending player | $1-20/month |
| F2P player | Non-spending player | 90-95% of players; content validates the game |
| battle pass | Monthly subscription-style IAP | ~$9.99/mo; best conversion tool |
| cosmetics | Non-gameplay skins/themes | High-margin, low regulatory risk |
| P2W | Pay-to-win | To be avoided; bad for western market reception |
| ARPDAU | Average Revenue Per Daily Active User | Target: $0.05–0.15 |
| LTV | Lifetime Value | Target paying player: $50–500+ over 12 months |
| conversion rate | % of F2P players who make any purchase | Target: 5-12% with battle pass |
| fair play pledge | Yada's ethical gacha brand commitment | Transparent odds, pity, F2P-completable content |
| MTX | Microtransactions | General term for in-game purchases |

## Platform / Tech Terms

| Term | Meaning | Context |
|------|---------|---------|
| HTML5 | Browser-native game technology | No download, works on any device |
| PWA | Progressive Web App | Enables push notifications for browser games; install-optional |
| cross-device sync | Cloud save across phone + desktop | Critical retention feature; g123 has it, Poki doesn't |
| SDK | Software Development Kit | Tools for third-party developers to build on Yada |
| dev platform | Yada's third-party developer ecosystem | Publisher/rev-share program for indie studios |
| rev share | Revenue sharing with game developers | Target: 70-80% developer / 20-30% platform |
