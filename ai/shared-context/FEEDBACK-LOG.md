## How to Add an Entry

Format:
```
**[YYYY-MM-DD] [Content type]:** [Specific correction in one sentence.]
Example: "Do not end banner descriptions with rhetorical questions — they read as weak."
```

Keep entries concrete and actionable. "Be better" is not an entry. "Never end a character's line with an ellipsis — it reads as passive" is an entry.
